HTML (HyperText Markup Language)

HTML is the language of structure. It uses elements (tags) to define content like headings, paragraphs, links, images, forms, and tables.

Page Skeleton

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Page</title>
  </head>
  <body>
    <h1>Heading</h1>
    <p>Paragraph text.</p>
  </body>
</html>

Semantic Elements

Use semantic tags to describe meaning, improving accessibility & SEO:

  • <header>, <nav>, <main>, <section>, <article>, <aside>, <footer>

Links & Images

<a href="https://example.com" target="_blank">Visit Example</a>
<img src="cat.jpg" alt="Playful cat" width="300">

Tip: Always include meaningful alt text for images.

Lists & Tables

<ul>
  <li>Item A</li>
  <li>Item B</li>
</ul>

<table>
  <thead><tr><th>Name</th><th>Role</th></tr></thead>
  <tbody><tr><td>Ada</td><td>Engineer</td></tr></tbody>
</table>

Forms

<form action="/submit" method="POST">
  <label>Email: <input type="email" name="email" required></label>
  <button type="submit">Send</button>
</form>

Client-side validation can be enhanced with JavaScript (see our homepage form).
